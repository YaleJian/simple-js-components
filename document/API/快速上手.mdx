使用simple web库，可以一次性获得全部能力。
## 安装

### 通过 npm 安装
在现有项目中使用 simple-web 时，可以通过 npm 进行安装
```bash
npm i simple-web
```
当然，你也可以通过 yarn 或 pnpm 进行安装：
```bash
# 通过 yarn 安装
yarn add simple-web

# 通过 pnpm 安装
pnpm add simple-web
```


### 通过 CDN 安装
使用 simple-web 最简单的方法是直接在 HTML 文件中引入 CDN 链接，之后你可以通过全局变量 sTools 访问到所有方法。

```javascript
<script src="https://unpkg.com/browse/simple-web@1.0.0-beta.1/dist/simple-web.js">
</script>
```
