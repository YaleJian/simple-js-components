# 按钮
按钮作为交互的最基础组件，不合适去个性化封装，更多从无损扩展的角度设计。

<button className="s">Button</button>

## 尺寸 size
```html
<button class="s xx-small">xx-small</button>
<button class="s x-small">x-small</button>
<button class="s">default</button>
<button class="s small">small</button>
<button class="s medium">medium</button>
<button class="s">large</button>
<button class="s x-large">x-large</button>
<button class="s xx-large">xx-large</button>
<button class="s xxx-large">xxx-large</button>
```
