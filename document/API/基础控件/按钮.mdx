# 按钮
按钮作为交互的最基础组件，不合适去个性化封装，更多从无损扩展的角度设计。

## 基础按钮
一个基础的按钮，只定义背景、边框、圆角，悬停、按下。
<button className="s">Button</button>

## 尺寸 size
```html
<button class="s xx-small">xx-small</button>
<button class="s">default</button>
```
<button className="s xx-small">xx-small</button>
<button className="s x-small">x-small</button>
<button className="s">default</button>
<button className="s small">small</button>
<button className="s medium">medium</button>
<button className="s large">large</button>
<button className="s x-large">x-large</button>
<button className="s xx-large">xx-large</button>
<button className="s xxx-large">xxx-large</button>

<button class="s black radius-1">b</button>

## 禁用 disabled
```html

```
