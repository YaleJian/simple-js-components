# 按钮
按钮作为交互的最基础组件，不合适去个性化封装，更多从无损扩展的角度设计。

## 基础按钮
1. 至少有以下属性和参考标准：
- 背景：通常为白色或者透明、暗黑模式下为深灰
- 边框：通常为1px，灰色
- 圆角：通常为6px
- 悬停：通常为变深、变亮
- 按下：通常为变得更深
2. 风格化时参考标准
- 按钮内的元素和当前行文字底部齐平
- 按钮内的矢量元素例如文字大小应该为1rem
- 按钮内应该有适当的留白
