# Element工具
扩展Element类功能的工具，所有组件都可以继承的功能。

## 使用方法
### 前置条件
根据Element类的语法要求，需要同时设置observedAttributes和attributeChangedCallback内置事件才能触发，工具才能完全生效。

#### 设置要监控的属性
在你的组件类重写observedAttributes属性，否则你的自定义属性值变化不会触发内置方法
```javascript
static get observedAttributes() {
    return [
        "disabled",
        "warning",
    ];
}
```
#### 定义属性回调方法
```javascript
attributeChangedCallback(name, oldValue, newValue) {
    //your code
}
```

### 调用
在需要使用Element工具的类中
```javascript
import stools from "simple-js-tools"

const {element} from stools

class example {
    //Your Class
}

element.init(example)

```
